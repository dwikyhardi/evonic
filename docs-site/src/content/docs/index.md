## Apa itu Evonic?

**Evonic** adalah platform AI agent orchestration yang powerful dan fleksibel. Dengan Evonic, kamu bisa:

- 🚀 **Kelola server** — start, stop, restart, dan monitor server Flask dengan mudah lewat CLI.
- 🔧 **Atur konfigurasi** — setup LLM provider, reconfigure supervisor, dan banyak lagi.
- 🧩 **Plugin & Skill** — install, enable, disable plugin, dan kelola skill dengan perintah sederhana.
- 🩺 **Diagnostik** — cek kesehatan sistem dengan `evonic doctor`.
- 🔄 **Self-update** — update Evonic otomatis dari CLI.

## Mulai

Jalankan perintah berikut untuk memulai:

```bash
./evonic setup
```

Atau langsung cek status server:

```bash
./evonic status
```

Lihat [panduan CLI](/cli/overview/) untuk daftar lengkap perintah.
